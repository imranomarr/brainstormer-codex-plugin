# Security Policy

Report suspected Brainstormer Codex plugin or MCP vulnerabilities privately to `imran@brainstormer.chat`.

Include the affected plugin or MCP version, a concise description, reproduction steps that do not expose another person's data, and the security impact. Do not include bearer tokens, OAuth authorization codes, refresh tokens, session content, database rows, or other private data.

Do not open a public GitHub issue for an unpatched vulnerability. General bugs that do not expose private data or weaken authorization may be reported through normal support channels.
